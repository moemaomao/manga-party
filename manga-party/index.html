Manga Party

Manga Party adalah tool hosting reader manga berbasis Blogger. Tool ini mengambil feed dari blog manga kamu lalu mengubahnya menjadi library manga lengkap dengan cover, detail, dan chapter reader.

Cocok untuk:
• Hosting manga di Blogger
• Tachiyomi source
• Backup library
• Reader statis tanpa backend
